# Derivative Creation Model - SLiM Simulation

## الميزات:
- بداية من آدم وحواء (زوج واحد)
- تتبع Y-chromosome و mtDNA
- حدث تضاعف الكروموسوم X
- قياس زمن الالتقاء والتنوع الأليلي
